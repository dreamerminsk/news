pages
Экранируемые имена страниц, раздёленные | символы, такие как Кот|Собака (по умолчанию)
Для связи с текущей страницей в вики, используйте {{FULLPAGENAMEE}}
range
Специальный диапазон для использования вместо точных дат start и end. Может быть одним из:
latest (По умолчанию) Данные за последние 20 дней
latest-N где N — число. Например, latest-30 будет показывать данные за последние 30 дней
current Последний доступный день
this-week Текущая неделя
last-week за прошлую неделю
this-month за этот месяц
last-month за прошлый месяц
this-year за этот год
last-year за прошлый год
all-time за всё время
start
Дата начала в формате YYYY-MM-DD.
Опустите этот параметр и параметр end, чтобы отобразить самые последние данные.
Использование формата YYYY-MM будет считаться стартовым месяцем и показывать ежемесячные данные вместо ежедневных.
Используйте earliest для самого раннего доступного дня.
end
Дата окончания в формате YYYY-MM-DD, по умолчанию — предыдущий день.
Использование формата YYYY-MM будет считаться конечным месяцем и показывать ежемесячные данные вместо ежедневных.
Используйте latest для последнего доступного дня.
platform
Одно из значений: all-access (По умолчанию), desktop, mobile-app, mobile-web
agent
Одно из значений: user (просматривавший человек, значение по умолчанию), spider (поисковый сканер), automated (автоматические программы) или all-agents (все типы)
redirects
Включает в себя просмотры всех перенаправлений запрошенных страниц. Установите 1 для включения или 0 для отключения (по умолчанию). Перенаправления могут отображаться принудительно, если в настройках пользователя выбран параметр «Всегда включать перенаправления». Также обратите внимание, что при наличии большого количества перенаправлений это может значительно замедлить работу инструмента.
autolog
Установите в false, чтобы отключить автоматическую настройку опции "Логарифмическая шкала"
mutevalidations
Установите параметр в true, чтобы не отображать сообщения при валидации формы. Значения по умолчанию будут применяться для некорректных значений, как обычно.
